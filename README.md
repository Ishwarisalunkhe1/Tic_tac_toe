

# Tic Tac Toe

Hello everyone! I’m **Ishwari Salunkhe**.

I’ve created a **Tic Tac Toe game** using basic web development technologies like **HTML, CSS, and JavaScript**.

It’s a **fun and interactive game** with **neon glowing animations** and a responsive design, making it enjoyable to play on both desktop and mobile screens.

If you enjoy playing it, **don’t forget to give it a like ⭐**!



## 🛠️ Tech Stack

This project is built with:

* **HTML** – For the structure of the game board and layout.
* **CSS** – For styling, neon animations, and responsive design.
* **JavaScript** – For game logic, player turns, win/draw detection, and interactive features.
* **Google Fonts** – *Permanent Marker* font for a fun handwritten style.


## 🎮 Features

* Neon glowing X and O tokens.
* Smooth pulsing animations for the grid and status text.
* Detects wins, draws, and updates the player turn automatically.
* Fully responsive design for different screen sizes.
* Restart button to play again instantly.



## 📝 How to Play
1. Click on any empty cell to place your **X** or **O**.
2. Players take turns until someone gets **three in a row** (horizontally, vertically, or diagonally).
3. If all cells are filled without a winner, the game ends in a **draw**.
4. Click the **Restart** button to play again.
